1. Next.js CSS Styles and Layout
2. Next.js Context API
3. Next.js API - MongoDB - Prisma Structure
4. Next.js Authentication (Auth.js) - Sign in with Google 
5. Next.js MongoDB Prisma (App Router)
6. Next.js SWR Mutation
